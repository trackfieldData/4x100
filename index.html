4x100 Relay Exchange Calculator — MVP

Math-first exchange point optimizer + coach go-mark adjustment. Uses your 100m 10m-split regression model.

Inputs
1.20 m (handoff geometry / reach)
0.12 s (outgoing starts moving after this)
30 m (college / pro)
How to use: enter estimated open 100m times. The optimizer chooses an exchange point in-zone to minimize baton time through the 30m zone (instantaneous exchange). Then use the Go mark adjustment sliders to trade a bit of time for safety.
Athlete 100m times
This MVP assumes each leg’s speed profile shape follows the same 100m model for their time. Legs 2–4 can be evaluated beyond 100m using a decel tail derived from the last three 10m segments.
Coach go-mark adjustment
Adjustment is applied to the optimal go mark (positive = outgoing starts earlier; negative = starts later).
